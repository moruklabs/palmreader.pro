# palmreader.pro

Static landing page for Palm Reader — AI palmistry iOS app.

## Deploy

Static files served from root. Legal/support/community URLs redirect through `moruk.link`.

## Files

- `index.html` — landing
- `robots.txt` — crawler allowlist
- `sitemap.xml` — search index map
- `app-ads.txt` — AdMob publisher verification (pub-9347276405837051)
